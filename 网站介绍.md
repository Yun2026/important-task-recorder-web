# 重要事务记录本 - 网站介绍

## 项目概述

重要事务记录本是一款面向个人用户的事务管理 Web 应用，旨在帮助用户高效规划、跟踪和完成日常任务。项目采用现代化的前端技术栈和云原生架构，实现了完整的事务管理生命周期，包括任务创建、分类管理、优先级排序、番茄钟专注和数据统计分析等核心功能。

## 技术架构

项目采用前后端分离的现代化架构。前端基于 Vue 3 + TypeScript 构建，使用 Vite 作为构建工具，Tailwind CSS 进行样式设计，TDesign Vue Next 作为 UI 组件库，确保了良好的用户体验和界面美观性。

后端采用腾讯云 CloudBase 无服务器架构，前端直接通过 CloudBase JavaScript SDK 连接云数据库，无需传统的后端服务器。这种架构降低了运维复杂度，提高了系统的可扩展性和稳定性。数据存储采用 CloudBase NoSQL 数据库，支持用户注册登录和任务数据的云端持久化，确保数据安全可靠。

## 核心功能

事务管理功能支持创建、编辑、删除任务，并提供优先级设置（高、中、低）和分类管理（工作、个人等），帮助用户清晰区分任务重要性和类别。番茄钟专注模式提供可自定义时长的倒计时功能，默认 25 分钟，帮助用户在处理复杂任务时保持专注。数据统计功能实时展示任务完成情况和时间分布，为用户提供直观的工作效率反馈。

界面采用响应式设计，完美适配桌面和移动设备，用户可以在任何设备上便捷使用。用户系统支持邮箱密码注册登录，结合 CloudBase 云数据库存储，确保用户数据安全隔离和多设备访问。

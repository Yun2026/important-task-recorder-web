# 📋 重要事务记录本 Web

基于 Vue 3 + TypeScript + Vite 的现代化任务管理 Web 应用，支持任务管理、专注模式、番茄钟、数据统计等功能。

## 🌐 在线访问

- **前端地址**: http://106.52.126.158:8080/

## ✨ 功能特性

- ✅ **任务管理** - 创建、编辑、删除任务，支持优先级设置和分类
- 🍅 **专注模式** - 番茄钟计时器，帮助提高工作效率
- 📊 **数据统计** - 任务完成统计、时间分布分析
- 🔐 **用户系统** - 登录/注册功能，本地存储用户数据
- 💾 **数据持久化** - 本地存储 + CloudBase 云存储
- 📱 **响应式设计** - 适配桌面和移动端

## 🚀 快速开始

### 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

### Docker 部署

```bash
# 构建并启动
docker-compose up -d

# 停止服务
docker-compose down
```

## 📖 项目介绍

重要事务记录本是一款专为提升个人工作效率而设计的现代化任务管理 Web 应用。在当今快节奏的工作环境中，有效管理时间和任务变得至关重要，这款应用正是为了解决这一痛点而生。

应用采用 Vue 3 + TypeScript 技术栈构建，界面简洁美观，交互流畅自然。核心功能围绕任务管理展开，用户可以创建、编辑、删除任务，为任务设置优先级和分类标签，让繁杂的工作变得井井有条。为了帮助用户更好地集中注意力，应用内置了专注模式，通过番茄钟技术帮助用户在工作与休息之间找到平衡，显著提高工作效率。

数据统计功能让用户可以直观地查看任务完成情况和时间分布，帮助发现工作习惯中的规律和改进空间。用户系统支持注册登录，数据可保存在本地或云端，确保重要信息不会丢失。整个应用采用响应式设计，无论是在电脑还是移动设备上都能获得良好的使用体验。后端部署在腾讯云轻量应用服务器上，通过 Docker 容器化技术实现快速部署和稳定运行，为用户提供持续可靠的服务。
